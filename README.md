# Artemis Program Website

A simple, student‑friendly website that explains NASA’s Artemis Program. It provides clear summaries of the Artemis I–IV missions, along with overviews of the Space Launch System (SLS) and the Orion spacecraft. Built using HTML, CSS, and JavaScript for use by the Sancta Space Club at Sancta Maria College.
